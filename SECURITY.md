# Security

These skills influence agent behavior when using XTrader MCP tools.

Rules:

- Keep this repository trusted and reviewed.
- Do not accept unreviewed changes to `SKILL.md` files.
- Do not add scripts unless they are necessary and audited.
- Treat legacy MCP writes as high risk.
- Treat graph replacement, dependency mutation, snapshot restore, and live/protected execution as high risk.
- Skills must guide agents to use MCP governance, validation, confirmation, and audit flows.
