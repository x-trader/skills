# Result Summarization

Summaries should be compact and decision-oriented.

Include:

- completion/progress state
- total symbols and processed symbols
- aggregate results and win-rate when available
- number of positions
- major issues
- most affected symbols
- notable positions
- high-signal logs only
- recommended next investigation steps

Avoid:

- raw payload dumps
- full logs
- full drawing lists
- all positions unless the user explicitly asks
- claims not supported by MCP result data
