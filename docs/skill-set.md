# XTrader MCP Skill Set

The first skill set contains nine MCP-only skills:

1. `xtrader-session`
2. `xtrader-catalog`
3. `xtrader-types`
4. `xtrader-nodes`
5. `xtrader-visual-graphs`
6. `xtrader-graph-plan`
7. `xtrader-governance`
8. `xtrader-backtests`
9. `xtrader-direct`

Scope: these skills guide agents that use XTrader MCP tools. They are not for non-MCP workflows.
